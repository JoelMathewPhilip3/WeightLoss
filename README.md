- build.gradle
- app/
- website/

## Hidden workflow file

GitHub does not let you upload hidden folders normally in some cases.

Create the workflow manually:

1. Go to your GitHub repository.
2. Click Add file.
3. Click Create new file.
4. In the filename box, type:

.github/workflows/build-apk.yml

5. Paste the workflow code provided by ChatGPT.
6. Commit changes.

## Build APK

1. Go to Actions.
2. Click Build WeightLoss APK.
3. Click Run workflow.
4. Download WeightLoss-debug-apk from the completed run.
